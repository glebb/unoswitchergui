Qt GUI for UnotellyCliSwitcher

Clone this repo.
Do "git submodule init && git submodule update" to fetch UnotellyCliSwitcher (https://github.com/VilleSalonen/UnotellyCliSwitcher).
Configure UnotellyCliSwitcher by creating switch.cfg to lib/UnotellyCliSwitcher

Make sure you have phantomjs and casperjs in your PATH.

Build:
qmake && make
